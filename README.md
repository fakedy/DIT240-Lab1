
WEBSERVER:
To run the webserver run http_server.go in a terminal and specify the port number that the webserver will use.

The webserver will only handle GET and POST any other requests will respond with appropiate response.
testing can be done either through curl requests or a webbrowser.

PROXY SERVER:
To run the proxy sever run proxy_server.go in a terminal specify the port number that the proxy server will use.

The proxy server will only handle GET any other requests will respond with appropiate response.
